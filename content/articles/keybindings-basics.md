## まず覚える5つ

| キー | 何が起こる |
|------|-----------|
| **Shift+Tab** | パーミッションモード切替 |
| **Esc** | 現在の生成を中断 |
| **↑** | 前のプロンプトを呼び出し |
| **Ctrl+C** | 実行中のコマンドを停止 |
| **Ctrl+L** | 画面クリア（会話は残る） |

これだけで日常の操作はほぼカバーできる。

## 全ショートカット一覧

### 入力関連

| キー | 動作 |
|------|------|
| `Enter` | プロンプト送信 |
| `Shift+Enter` | 改行（送信しない） |
| `↑` / `↓` | プロンプト履歴の移動 |
| `Esc` | 生成を中断 / 入力をクリア |

### モード切替

| キー | 動作 |
|------|------|
| `Shift+Tab` | パーミッションモードを順に切替 |
| `Alt+M` | パーミッションモード切替（別キー） |

### ファイル・画像

| キー | 動作 |
|------|------|
| ドラッグ&ドロップ | ファイルや画像を添付 |
| パスを貼付 | ファイルパスを直接入力 |

## カスタマイズしたいとき

```
キーバインドをカスタマイズしたい。
~/.claude/keybindings.json の設定方法を教えて。
```

Claude Codeに聞けば設定ファイルの書き方を教えてくれる。
