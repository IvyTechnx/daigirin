```
ドラッグ&ドロップでCSVを読み込んで、
テーブル表示、ソート、フィルタリングができるように。
Next.jsで。
```

## CSV変換ツール

```
CSVのカラムを変換するツールを作って。
入力: 元のCSV
出力: 指定したカラムだけ抽出 + カラム名変更
設定はUIで指定できるように。
```

## 集計ダッシュボード

```
売上データのCSV（日付, 商品名, 金額, カテゴリ）を
アップロードしたら、以下のダッシュボードを表示するアプリ作って:
- 月別売上グラフ
- カテゴリ別の円グラフ
- トップ10商品
rechartsでグラフ描画して。
```

## Excelの定型作業を自動化

```
毎月やってる作業:
1. CSVをダウンロード
2. 特定のカラムだけ抽出
3. 日付をYYYY/MM/DD形式に変換
4. 金額に3桁カンマをつける
5. 新しいCSVで出力

これを自動でやるスクリプト作って。Node.jsで。
```

## 「毎月やる作業」をまるごと自動化

```
毎月1日にやってる報告作業:
（作業手順を貼り付け）

これを1コマンドで全部やるスクリプト作って。
```

Claude Codeは業務手順書をそのまま読んで自動化スクリプトに変換できる。
