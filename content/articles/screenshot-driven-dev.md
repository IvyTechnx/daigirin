## やり方

1. 作りたいUIのスクショを撮る（Webサイト、アプリ、手書きメモでもOK）
2. Claude Codeに貼る
3. 「これ作って」

以上。

## 実例

### パターン1: 参考サイトをコピー

```
（Notionのスクショを貼付）

このUIをNext.jsで再現して。
左にサイドバー、右にコンテンツエリアのレイアウトで。
```

### パターン2: 手書きワイヤーフレーム

```
（紙に書いたラフスケッチを撮影して貼付）

このラフの通りに画面を作って。
ヘッダー、3カラムのカード、フッターの構成で。
```

### パターン3: 既存画面の修正

```
（自分のアプリのスクショを貼付）

この画面のここが気に入らない。
- ボタンをもっと目立たせて
- カード間のスペースを広げて
- フォントサイズを1段階上げて
```

### パターン4: 別アプリの一部を取り入れる

```
（Spotifyのプレイリスト画面のスクショを貼付）

このリスト表示のスタイルを参考に、
うちのアプリの一覧画面をリデザインして。
完全コピーじゃなくて雰囲気だけ寄せて。
```

## なぜスクショが効くのか

- 言葉で「いい感じのカード」と言っても曖昧
- スクショなら余白、配色、レイアウトが一発で伝わる
- Claude Codeは画像を理解して、具体的なCSSに落とし込んでくれる

## コツ

- **全体スクショ + 部分スクショ**を両方送ると精度UP
- 「この部分だけ真似して」と範囲を指定すると暴走しない
- 色やフォントにこだわりがあれば言葉で補足する

```
このデザインを再現して。
ただし色は紫系（#6D28D9）に統一して。
フォントはGeistで。
```

## 注意点

- 著作権のあるデザインの完全コピーは避ける
- あくまで「参考」「雰囲気」として使う
- ロゴやブランド要素はスクショに含めない
