## 全体を整理

```
このプロジェクトのコードを整理して。
重複してる処理をまとめて、
ファイル構成を整理して。
動作は変えないで。
```

## 特定ファイルのリファクタリング

```
src/app/page.tsx が長すぎる。

```

## JavaScript → TypeScript

```
このプロジェクトをTypeScriptに変換して。

型定義をちゃんとつけて。
```

```

```
